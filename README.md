# form-token
inserts a form token in each form wchich method is post method.

# Example

package.json

```javascript
{
  "dependencies": {
    "@linkorb/form-token": "^0.1",
  }
}
```

app.js:
```javascript
  import '@linkorb/form-token';
```

### Define `formToken` global variable in file

```javascript
  var formToken = "xxx-xx-xxx";
```
